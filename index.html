What is Css?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. Css is not necessaryCss saves a lot of work. It can control the layout of multiple web pages all at once.

  4. External style sheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using the style element in the head section.
  • External - by using a link element to link to an external CSS file
  • The most common way to add CSS, is to keep the CSS files. However, in this 
    tutorial we will use inline and internal styles,     beacuse this is easier to demonstrate, and easier 
    for you to try it yourself
When comparing Css class vs ID, the difference is that css applies a style to multiple elements.ID on the other hand, applies a style to oen unique element. ID is also
special in that you can use a special url to link directly to an element and it is used by javascript.
In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the
element
There is a wide range of selectors avaliable. Two of the most commonly used are class and ID. both are used to target elements to which a style should be applied.
